# Папка `droplets` (Photoshop)

Положите **рядом с `AutoRAW.exe`** в каталог **`droplets\`** (или **`droples\`**, если так названа папка у вас) сохранённые из Photoshop экшен-дроплеты с такими **именами файлов**:

| Файл | Кадры (номер экспорта) |
|------|-------------------------|
| `01_drop.exe` | 01 |
| `02-03-04-08_drop.exe` | 02, 03, 04, 08 |
| `05-06-07_drop.exe` | 05, 06, 07 |

В главном окне включите блок **«Прогнать через Дроплеты»** (по умолчанию выключено). После **полной записи** каждого кадра пакета (WebP или JPEG) AutoRAW запускает соответствующий `.exe` и передаёт **полный путь** к сохранённому файлу первым аргументом командной строки (как при перетаскании на дроплет в Проводнике).

Папку можно скопировать вместе со сборкой или создать вручную. При **`build-installer.ps1`** в установщик попадает всё содержимое **`droplets\`** (и **`droples\`**, если есть) из корня репозитория на машине сборки — положите три `.exe` в `droplets\` перед сборкой. В git бинарники дроплетов не коммитятся (см. `.gitignore`).
