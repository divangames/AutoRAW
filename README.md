# AutoRAW

**AutoRAW** — десктопное приложение для Windows: **пакетная выгрузка** снимков (в т.ч. RAW) в размер **референса** с **ручной** подгонкой кадра (редактор положения / `manual_shot_adjust.json`) поверх **полного** исходника после ориентации по номеру кадра. Папка **`zona`** используется в основном для **PNG сетки** в редакторе; автоматический кроп по красному маркёру в пакете **отключён**. Стек: **.NET 8**, **WPF**, **ImageMagick**, **OpenCvSharp**.

Команда: [**WhiteCube**](https://белаякоробка.рф/) · репозиторий: [github.com/divangames/AutoRAW](https://github.com/divangames/AutoRAW)

---

## Технология «Zona» (в профиле)

В **текущей версии** пакет **не** режет кадр по маркёру Zona: итог задаётся **референсом** (размер файла) и **ручными** правками. Каталог **`zona`** по-прежнему удобен для шаблонов **сетки выравнивания** в **редакторе положения кадра** и для совместимости структуры профиля.

Историческое описание маркёров и макетов — в **`src/AutoRAW/Assets/Instruction.md`** (копия рядом с exe после сборки).

## Возможности

- **Профили товара** — меню **Профиль → Стандартные** (`profiles/<slug>/profile.json`, `reference` и `zona` рядом с программой) и **Пользовательские** (пакеты в `%LocalAppData%\AutoRAW\user files\Profile`); импорт своего профиля — **одной папкой**.
- **Пакетная обработка** — рекурсивный обход вложенных папок во входе «Товар», сохранение **той же структуры подпапок** в выходе; **пауза** и **отмена**; ошибки по файлам не останавливают весь пакет.
- **Чат Zona** — журнал в виде переписки с **ZONA** (пузыри, аватар, копирование); отдельное окно журнала.
- **Темы** — светлая, тёмная и **как в системе**: меню **Вид → Тема**; настройки в `%AppData%\AutoRAW\theme_prefs.json`.
- **Telegram** — меню **Настройки → Telegram ZONA…**: уведомления о **завершении пакета**, **ошибке** и **отмене**; токен и chat id сохраняются на ПК; смена учётных данных — через **«Изменить…»** и пароль.
- **Простой и расширенный режим**, **drag-and-drop** папки или файла во вход «Товар».
- **Цветокоррекция по XMP** (альфа) с превью «до / после».
- **Экспорт WebP или JPEG** с запоминанием выбора.
- **Обновления с GitHub** — **Справка → Проверить обновление** и напоминание при старте, если на [Releases](https://github.com/divangames/AutoRAW/releases) есть более новая версия (нужен **`AutoRAW-Setup-*-ru.exe`** во вложениях релиза).

Актуальный список изменений — в **[CHANGELOG.md](CHANGELOG.md)**. Версия в программе — **пятикомпонентная** (`0.7.8.1.0`): смысл позиций и правила записи описаны в начале changelog; в UI и в имени установщика используются все пять чисел, в манифесте Windows — первые четыре.

## Требования

### Программное обеспечение

- **Windows 10 / 11**, только **x64**
- **.NET 8 Windows Desktop Runtime** (для portable-сборки; установщик при необходимости ставит зависимости)

### Минимальные требования к ПК

Подойдёт для одиночных файлов и небольших пакетов (JPEG/WebP, умеренное разрешение).

| | |
|---|---|
| **Процессор** | 2 ядра, x64, ~1,5 ГГц и выше |
| **Оперативная память** | **4 ГБ** |
| **Место на диске** | ~1 ГБ под программу, кэш и временные файлы + объём ваших исходников и результата |
| **Носитель** | HDD или SSD |
| **Экран** | от **1280×720** (часть окон и чата удобнее на более широком окне) |
| **Видеокарта** | Достаточно **встроенной графики** процессора; дискретная не обязательна |

### Рекомендуемые требования

Комфортная работа с **пакетами RAW**, большим разрешением, рекурсивными папками и превью «до / после».

| | |
|---|---|
| **Процессор** | **4 ядра и больше** (или 6+ потоков), современные архитектуры Intel/AMD |
| **Оперативная память** | **8 ГБ** и больше; для очень больших серий **16 ГБ** |
| **Место на диске** | **SSD**; свободного места с запасом **в 2–3 раза** больше, чем суммарный объём входных файлов и экспорта |
| **Экран** | **1920×1080** и выше |
| **Сеть** | Для **обновлений с GitHub** и **уведомлений Telegram** — доступ в интернет (остальная работа офлайн) |

## Сборка и запуск (разработка)

- Установите [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) и при необходимости **Inno Setup 6** (для `build-installer.ps1`).
- В корне репозитория: **`dev.cmd`** — меню (сборка, запуск, установщик).
- В папку `installer` при сборке установщика положите бандл **`dotnet-sdk-*-win-x64.exe`** для Inno (файл в `.gitignore`, в репозиторий не коммитится).

---

## Установщик

Скрипт **`build-installer.ps1`** публикует приложение и собирает **Inno Setup** (`installer\AutoRAW.iss`). Версия читается из **верхней** строки **`CHANGELOG.md`** (`## [a.b.c.d.e]`, допускается 1…5 чисел — недостающие справа нули). Скрипт **`bat\Resolve-VersionFromChangelog.ps1`** пишет: **полная строка** в `MyAppVersion` (тексты мастера и имя `AutoRAW-Setup-…-ru.exe`) и **первые четыре компонента** в `MyAppVerFull` (поле VersionInfo). Результат: `dist\AutoRAW-Setup-0.7.8.1.0-ru.exe` (папка `dist/` в git не входит).
