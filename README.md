# AutoRAW

**AutoRAW** — десктопное приложение для **пакетного кадрирования** снимков (в т.ч. RAW) по **референсу** или по **технологии «Zona»** — собственному режиму кропа по красному маркёру на парных изображениях (см. раздел ниже). Стек: **.NET 8**, **WPF**, **ImageMagick**, **OpenCvSharp**.

Команда: [**WhiteCube**](https://белаякоробка.рф/) · репозиторий: [github.com/divangames/AutoRAW](https://github.com/divangames/AutoRAW)

---

## Технология «Zona»

**«Zona»** — это **собственная разработка для AutoRAW**, а не сторонний универсальный плагин: кадрирование по маркёрному изображению с **красной зоной**, детекция контура, перенос геометрии на полноразмерный снимок и вырез с учётом поворота. В профиле используется каталог с именем `zona`; при совпадении имён с входным файлом и успешном распознавании маркёра **Zona** имеет приоритет над кропом по `reference`.

Подробное описание и обновления формулировок — в **`src/AutoRAW/Assets/Instruction.md`** (раздел «Технология Zona»; копия рядом с exe после сборки).

## Возможности

- **Простой и расширенный режим** — папка «Товар», опционально выход; таблица соответствий и ручные пути в расширенном режиме.
- **Профили товара** — встроенный профиль и пользовательские пакеты (`reference`, `zona` для **Zona**, `setting` с XMP); меню **Профиль**, каталог профилей в `%LocalAppData%\AutoRAW\user files\Profile`.
- **Цветокоррекция по XMP** (альфа) с превью «до / после».
- **Экспорт WebP или JPEG** с запоминанием выбора.
- **Проверка обновлений** — меню **Справка → Проверить обновление** и напоминание при старте, если на [GitHub Releases](https://github.com/divangames/AutoRAW/releases) есть более новая версия (нужен установщик **`AutoRAW-Setup-*-ru.exe`** во вложениях релиза).

## Требования

- **Windows 10+**, x64  
- **.NET 8 Windows Desktop Runtime** (для portable-сборки; установщик при необходимости ставит зависимости)

---

## Сборка и запуск (разработка)

- Установите [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) и при необходимости **Inno Setup 6** (для `build-installer.ps1`).
- В корне репозитория: **`dev.cmd`** — меню (сборка, запуск, установщик).
- В папку `installer` положите бандл **`dotnet-sdk-*-win-x64.exe`** для Inno (файл в `.gitignore`, на GitHub не хранится из‑за размера).

---

## Установщик

Скрипт **`build-installer.ps1`** публикует приложение и собирает **Inno Setup** (`installer\AutoRAW.iss`). Номер версии для установщика берётся из **верхней** записи `## [x.y.z]` в **`CHANGELOG.md`** (через `bat\Resolve-VersionFromChangelog.ps1`) и передаётся в ISCC как `/DMyAppVersion` и `/DMyAppVerFull`. Итог: `dist\AutoRAW-Setup-<версия>-ru.exe`. Не компилируйте `.iss` вручную из Inno IDE без этих define — в скрипте останутся заглушки `0.0.0`.

---

## Релизы и версии

- Версия сборки приложения и установщика берётся из **верхней** секции `## [x.y.z]` в **`CHANGELOG.md`** (цель MSBuild `ResolveVersionFromChangelog` и тот же разбор в `build-installer.ps1`).
- Публикация релиза на GitHub: тег вида **`v0.7.0`**, в описание — текст «что нового»; во вложения — **`AutoRAW-Setup-0.7.0-ru.exe`** (имя должно начинаться с `AutoRAW-Setup-` и заканчиваться на `-ru.exe`).
