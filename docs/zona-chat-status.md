## Что уже сделано

- Журнал переименован в UI: **«Чат Zona»** (меню, заголовок, отдельное окно).
- Сообщения — пузыри (не TextBox), `ListBox` + `StackPanel` (не Virtualizing — иначе не работает перенос).
- Telegram-подобная палитра dark: фон `#17212B`, входящие Zona `#2B5278`, системные `#182533`.
- Статусы по типу: Done — зелёный, Error/Cancel — красный, Pause — жёлтый оттенок.
- Время в пузыре справа снизу (`TimeShort`, формат `HH:mm`); в копировании — полная строка `[HH:mm:ss] …`.
- Круглый аватар ZONA (`FromZona`), пауза/продолжение/ошибки Zona с аватаром и нужным `LogLineKind`.
- Кнопка копирования 📋 36×36, появляется при наведении.
- `ZonaMessages` — рандомные фразы ZONA (100 вариантов, колода без повторов).
- Перетаскивание шапки → отдельное окно; автоскролл к новым сообщениям.

## Известные ограничения / не довели

- **Жирный `**текст**` у ZONA** — убран ради `Emoji.Wpf` (сейчас только `Body` без разметки). Нужно совместить bold + цветные emoji, если важно.
- **Хвостик пузыря** (треугольник как в Telegram) — только скругление `12,12,12,4`, без Path.
- **Время в одной строке** с коротким текстом (как в TG) — сейчас всегда под текстом в углу.
- **Цвета статусов** — пользователь просил ярче красный/жёлтый/зелёный; последняя итерация — ближе к референсу TG (приглушённые). При возврате уточнить: TG-референс vs. яркие цвета из ТЗ.
- **Светлая тема** — базово есть, чат под TG dark настраивался точнее.

## Telegram-уведомления (добавлено)

- Меню **Настройки → Telegram ZONA…** — токен бота, ID чата, вкл/выкл, кнопка «Проверить».
- Хранение: `%AppData%\AutoRAW\telegram_zona.json`.
- Отправка при: **готово** (пакет + статистика), **ошибка** (исключение), **отмена** (кнопка отмены).
- **Шаблон «готов пакет»:** без строки «ZONA — пакет готов»; порядок: фраза из `ZonaMessages.NextDone()`, пустая строка, **✅ Успешно: … Время: …**, строка **Профиль: …** (`MainViewModel` → `NotifyTelegram`).
- Сервисы: `ZonaTelegramNotifyService`, `ZonaTelegramPreferenceStore`.

## При возврате — логичные следующие шаги

1. Сверить UI с референсом Telegram (скрин пользователя): пузырь, аватар, время, отступы.
2. Решить: яркие статус-цвета (ТЗ) или спокойные (TG).
3. Bold + emoji (RichText / FlowDocument / Emoji.Wpf RichTextBox).
4. Опционально: хвостик пузыря, inline-время для коротких сообщений.
5. Подготовка к интеграции с Telegram (отдельный этап).
