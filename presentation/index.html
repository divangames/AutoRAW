Возможности

Всё для продуктивной работы с фото

Инструменты, которые превращают часы ручной обработки в минуты автоматики

Пакетная обработка

Рекурсивный обход всех вложенных папок. Структура подпапок во входе повторяется в выходе. Сбой одного файла не останавливает пакет — остальные обрабатываются.

Вложенные папки Пауза / Отмена Прогресс

Профили товара

Встроенные профили «Кроссовки» и «WB Sneakers». Создавайте свои — любые референсы и маркёры Zona.

Drag-and-drop

Перетащите папку или любой файл из неё прямо в поле «Товар (вход)» — путь подставится автоматически.

Темы интерфейса

Светлая, тёмная или автоматически как в системе Windows. Переключение без перезапуска — меню «Вид → Тема».

Цветокоррекция XMP

Применение настроек из XMP-файлов (экспозиция, контраст, насыщенность) с превью «до/после» в реальном времени.

Экспорт WebP / JPEG

Выбор формата сохраняется. Готовые файлы — рядом с входной папкой в `webp` или `jpg`, либо в заданный каталог.

Превью и анализ

Референс · До · После — три панели с живым пересчётом. Ползунок анализа регулирует точность кропа по длинной стороне.

Обновления с GitHub

Автоматическая проверка при запуске и ручная через меню «Справка». Загрузка и тихая установка нового установщика.

Технология «Zona»

Кроп по красному маркёру

Собственная разработка AutoRAW: рядом с каждым исходником лежит маркёрное фото с красной рамкой вокруг нужной области. Алгоритм находит контур, вычисляет геометрию вместе с поворотом и точно вырезает кадр.

01
Маркёрное фото
В папке zona/ — снимок с красной рамкой, имя файла совпадает с исходником.
02
Детекция контура
OpenCV находит красную область, строит ограничивающий прямоугольник с учётом угла поворота.
03
Перенос геометрии
Координаты масштабируются на полноразмерный RAW; вырезается финальный кадр нужного размера.
04
Приоритет над референсом
Если маркёр найден — Zona в приоритете. Иначе — автоматически переключается на кроп по референсу.
